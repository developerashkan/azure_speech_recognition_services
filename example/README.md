## Azure configuration

The example app reads Azure Speech credentials from `assets/azure.env`.

1. Open `assets/azure.env`.
2. Replace `YOUR_AZURE_SPEECH_TOKEN` and `YOUR_AZURE_SPEECH_REGION` with your real values.
3. Run `flutter pub get` (or restart the app) so Flutter bundles the asset.
